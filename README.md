## 📂 Repository Structure

### Exercises
Structured Python practice files organized by topic:
- **Functions** → Function definitions, parameters, return values
- **Loops** → For loops, while loops, loop control
- **Conditionals** → If-else statements, nested conditions
- **Data Structures** → Lists, dictionaries, sets, tuples
- **String Operations** → String methods, formatting, manipulation
- **File Handling** → Reading, writing, file operations

### Mini Projects
Real-world projects applying core concepts:
- Snake-Water-Gun Game
- Calculator
- File Processing Utilities

## ▶️ How to Run

Clone the repository:
```bash
git clone https://github.com/bhoomijn/python-basics-to-projects.git
cd python-basics-to-projects
```

Run any Python file:
```bash
python filename.py
```
